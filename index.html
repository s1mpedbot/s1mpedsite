s1mped — Telegram-бот

Экономический бот для Telegram. Без рекламы и доната.

О проекте

Экономическая система с элементами RPG внутри Telegram. Не требует дополнительных приложений или установок.

Постепенное открытие возможностей по мере развития. Подходит для личного использования и поддержания активности в чатах.

Основные возможности

Экономическая система

Несколько типов валют: димпы (основная), опыт, мандарины (временная), сердца.

Уровневая прокачка

Уровни от 1 до бесконечности. Уникальные звания до 250-го уровня.

Магазин

Покупка описаний профиля, валют, апгрейдов, VIP-статуса.

Социальные функции

Брачная система, бизнес, анонимные сообщения, переводы между пользователями.

Ежедневная активность

Команды .стрик, .трек, .заказ, .мульти с системой ожидания.

Боксы

Много интересных боксов для открытия. Коллекция карт.

Групповые функции

Управление группами, РП-команды, система поинтов для чатов.

И многое другое!

Нововведения

Добавлены ежедневные активности и системные изменения, направленные на более стабильную и честную прокачку.

Ежедневные активности

  • Личная активность (.стрик)
    Базовая ежедневная награда для всех пользователей: димпы, опыт, карты, шанс роста брака и сердца, доход от бизнеса.
  • Семейная активность (.трек)
    Доступна для пользователей с уровнем брака 25+. Награды: димпы на общий счёт, опыт, поинты отношений, сердца партнёру.
  • Бизнес-активность (.заказ)
    Для владельцев бизнеса с уровнем 20+. Доход на карту, шанс найма или потери сотрудников, четыре типа заказов.

Механика ожидания

Все активности используют общий кулдаун. Чем выше уровень апгрейда, тем меньше время ожидания.

Изменения и улучшения

  • Добавлен раздел «Помощь» — вся информация теперь доступна прямо в меню бота.
  • Опыт (XP) больше нельзя купить в магазине. В дальнейшем будет добавлен альтернативный способ получения.
  • Добавлены поинты группы — начисляются за активность в чатах.

Новая активность

Добавлена дополнительная активность для опытных пользователей с уровнем 50+:
/multi. Доступны все типы наград.

Технические особенности

  • Использует SQL для хранения данных
  • Модульная архитектура кода
  • GIF в РП командах
  • Есть темы для оформления, возможно создать свою
  • Московское время для всех временных меток
Весь функционал доступен бесплатно. Платные функции отсутствуют.

Начало работы

Для использования бота необходим аккаунт в Telegram.

  1. Перейдите по ссылке на бота
  2. Нажмите кнопку "Start" или отправьте команду /start
  3. Система автоматически создаст ваш профиль
  4. Используйте команду /menu для навигации
  5. Читайте описание команд через /help
